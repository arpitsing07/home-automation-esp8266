# Home Automation System (ESP8266 + Blynk)

## Overview
This project allows users to control electrical appliances using a mobile app and manual switch through Wi-Fi.

## Features
- Remote control via Blynk app
- Manual switch control
- Real-time status monitoring
- Low-cost IoT solution

## Components Used
- ESP8266
- Relay Module
- Blynk App
- Wi-Fi Router

## Working Principle
The ESP8266 connects to Wi-Fi and communicates with the Blynk server to control connected appliances.

## Outcome
Successfully built and tested working prototype for academic project.
